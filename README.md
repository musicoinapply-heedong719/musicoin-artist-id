## Overview

The Musicoin 3.0 Artist Portal is a gateway for artists to join and manage
their presence in the ecosystem. It focuses on:

- Easy onboarding
- Clear profile management
- Linking on-chain identity with off-chain channels

Each artist can register:

- Wallet address
- Official homepage URL
- Twitter/X handle and other social links
- Basic profile and branding

## Core Goals

- Make it simple for artists to join Musicoin 3.0
- Provide a canonical profile that other services (player, sampling, ticketing)
  can reference
- Ensure that artist information is consistent and easy to verify

## Key Features

- Artist registration:
  - Wallet address
  - Name / stage name
  - Homepage
  - Twitter/X and other social links
- Profile editing and verification
- Integration with:
  - Web Player (artist info display)
  - Sampling Studio (author attribution)
  - Ticketing & Instrument History

## Architecture

- **Smart Contract**:
  - Artist registry with minimal essential data
- **Frontend**:
  - Web UI for artists to register and manage profiles
- **Backend**:
  - Optional indexing and search service

## Roadmap

1. MVP: artist registration + basic profile
2. Public artist directory with search and filter
3. Advanced verification (e.g. social proof, signed messages)
4. Integration hooks for partner services
